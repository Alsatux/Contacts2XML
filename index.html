Contacts2XML

Version 0.4 - with JPG photo export

N.B.:

This webapp backups all your existing contacts on your SD-card, into the path contacts2xml. It adds a second .bak file for more security with Cockatoo, an external editor to manage your Firefox OS contacts !

Unplug any USB wire before doing a backup or you won't be able to write on the sd-card !

For more security, don't forget to copy all backup files outside of your phone, using the USB capabilities of Firefox OS.

To restore your contacts, please use the XML2Contacts webapp.
